# Deploy Headphones Page on Vercel

Repo name suggestion: `headphones-under-100`

## GitHub → Vercel steps

1. Create a new GitHub repo named `headphones-under-100`.
2. Upload the contents of this `headphones/` folder, especially `index.html`.
3. Go to Vercel → **Add New → Project**.
4. Import the `headphones-under-100` GitHub repo.
5. Framework preset: **Other** / static site.
6. Build command: leave blank.
7. Output directory: leave blank or `.`.
8. Click **Deploy**.
9. Open the Vercel URL and test on desktop + phone.
10. Replace placeholder `href="#"` CTA links with tracked affiliate links when ready.

## Static site settings

- Root directory: repo root
- Build command: none
- Output directory: `.`
- Install command: none
